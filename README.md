# 로또 번호 추첨기 (Lotto Number Generator)

현대적이고 깔끔한 UI를 가진 웹 기반 로또 번호 추첨기입니다.

## 주요 기능
- **번호 생성**: 1부터 45까지의 숫자 중 중복 없이 6개의 번호를 무작위로 추출합니다.
- **번호 정렬**: 추출된 번호를 오름차순으로 자동 정렬하여 표시합니다.
- **시각화**: 번호 대역별로 다른 색상의 공(Ball) 디자인을 적용했습니다 (1-10: 노랑, 11-20: 초록, 21-30: 파랑, 31-40: 빨강, 41-45: 회색).
- **초기화**: 버튼 하나로 간편하게 추첨 결과를 초기화할 수 있습니다.
- **반응형 디자인**: 모바일과 데스크탑 환경 모두에서 최적화된 화면을 제공합니다.

## 기술 스택
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
